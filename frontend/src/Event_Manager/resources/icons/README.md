# Icons Directory

This directory contains UI icons for the Event Manager application.

## Icon Files:
- `logo.png` - Application logo (64x64 or 128x128)
- `create_event.png` - Icon for create event button (24x24)
- `view_events.png` - Icon for view events button (24x24)

## Icon Guidelines:
- Use PNG format for transparency support
- Recommended sizes: 16x16, 24x24, 32x32, 64x64
- Keep file sizes small for better performance
- Use consistent style across all icons
