# Images Directory

This directory contains larger images for the Event Manager application.

## Image Files:
- `background.jpg` - Application background image
- `placeholder.png` - Placeholder image for events without photos

## Image Guidelines:
- Use JPEG for photos (smaller file size)
- Use PNG for graphics with transparency
- Keep reasonable dimensions (e.g., 300x200, 400x300)
- Optimize images for web to reduce file size
