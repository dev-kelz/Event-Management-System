# User Guide

Welcome to the Event Manager user guide. This guide will help you get started with using the Event Manager application.

## Getting Started

### Installation

1. **Download** the Event Manager application
2. **Install** the required dependencies
3. **Run** the application

### First Launch

When you first launch Event Manager, you'll see the main window with options to:
- Create new events
- View existing events
- Manage your event calendar

## Using the Desktop Application

### Creating Events

1. Click the **"Create Event"** button
2. Fill in the event details:
   - **Title**: Give your event a descriptive name
   - **Description**: Add details about the event
   - **Start Date/Time**: When the event begins
   - **End Date/Time**: When the event ends
   - **Location**: Where the event takes place
3. Click **"Save"** to create the event

### Viewing Events

The main screen shows your events in a list format. You can:
- **Browse** through all your events
- **Filter** events by date or location
- **Search** for specific events
- **Sort** events by date, title, or location

### Editing Events

1. Select an event from the list
2. Click **"Edit"** or double-click the event
3. Modify the event details
4. Click **"Save"** to update the event

### Deleting Events

1. Select an event from the list
2. Click **"Delete"**
3. Confirm the deletion

## Tips and Tricks

- **Keyboard Shortcuts**: Use Ctrl+N for new event, Ctrl+S to save
- **Quick Search**: Start typing to search events by title
- **Date Navigation**: Use the calendar widget to jump to specific dates
- **Backup**: Your events are automatically saved to the backend

## Troubleshooting

### Common Issues

**Application won't start:**
- Check that all dependencies are installed
- Ensure the backend server is running

**Can't create events:**
- Verify you're connected to the backend
- Check your internet connection

**Events not showing:**
- Refresh the event list
- Check the date filters

### Getting Help

If you encounter issues:
1. Check this user guide
2. Review the error messages
3. Contact support with details about the problem
